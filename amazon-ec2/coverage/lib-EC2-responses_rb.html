 11   # There are two major types of responses that can come back from EC2#method calls:
 12   #
 13   # * A 'Set' Array which is a collection of objects
 14   # * A 'Response' object (Response or Subclasses) which encapsulates data returned 
 15   #   as a result of running the method.  In this case the object is an OpenStruct 
 16   #   and is Enumerable so you can iterate over all of the data returned.
 17   # * Note regarding Boolean responses.  Those EC2 methods that simply respond with a 
 18   #   'true' when called will return an empty sub-class of the Response object that
 19   #   matches the calling method.  The presence of this object (as opposed to an Exception)
 20   #   means that that method call succeeded.
 21   #
 22  
 23   # The make_request() and ec2_error? methods, which are shared by all, will raise any 
 24   # exceptions encountered along the way as it converses with EC2.
 25   #
 26   # Exception Handling:  A call to any of these methods will respond with one of the
 27   # types mentioned above.  If for some reason an error occurrs when executing a method
 28   # (e.g. its arguments were incorrect, or it simply failed) then an exception will 
 29   # be thrown.  The exceptions are defined in exceptions.rb as individual classes and should
 30   # match the exceptions that Amazon has defined for EC2.  If the exception raised cannot be
 31   # identified then a more generic exception class will be thrown.

 33   # The implication of this is that you need to be prepared to handle any exceptions that

 38   # The superclass for all general responses.  A Set (or one of its sub-classes)
 39   # may return an Array of these objects.
 40   #
 41   # I learned the magic of making an OpenStruct object able to respond as a fully Enumerable 
 42   # object (responds to .each, etc.) thanks to a great blog article on Struct and OpenStruct 
 43   # at http://errtheblog.com/post/30
 44   #
 45   class Response < OpenStruct
 46     
 47     include Enumerable
 48     
 49     def members
 50       methods(false).grep(/=/).map { |m| m[0...-1] }
 51     end
 52     
 53     def each
 54       members.each do |method|
 55         yield send(method)
 56       end
 57       self
 58     end
 59     
 60     def each_pair
 61       members.each do |method|
 62         yield method, send(method)
 63       end
 64       self
 65     end
 66      
 67     def [](member)
 68       send(member)
 69     end
 70     
 71     def []=(member, value)
 72       send("#{member}=", value)
 73     end

 75   end
